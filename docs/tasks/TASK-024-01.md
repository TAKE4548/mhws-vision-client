# TASK-024-01: 依存関係のインストールと npm script の追加

## 概要
テスト環境構築の第一歩として、必要なライブラリをインストールし、`package.json` にテスト実行用のスクリプトを追加します。

## 完了条件
- [ ] `vitest`, `jsdom`, `@testing-library/react`, `@testing-library/jest-dom` が `devDependencies` に追加されている。
- [ ] `package.json` の `scripts` に `"test": "vitest"` が追加されている。

## 対象ファイル
- `package.json`

## 実装上の注意
- ライブラリのバージョンは現在のプロジェクトの Vite バージョン (5.1.0) や React バージョン (18.2.0) と互換性のあるものを選択する。
- `npm install -D vitest jsdom @testing-library/react @testing-library/jest-dom` を実行する。
