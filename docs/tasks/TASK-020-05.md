# TASK-020-05: [BUG] ROICalibrator.tsx への Sync Enabled トグルの追加

## 概要
ROI調整（Equal spacing sync）の有効/無効を切り替えるトグルを `ROICalibrator.tsx` に追加し、ユーザーが個別のROI微調整を行えるようにする。

## 状況
- `roiStore.ts` には既に `isSyncEnabled` 状態と `setSyncEnabled` アクションが存在する。
- しかし、`ROICalibrator.tsx` のUI上にこれを操作する手段がないため、常に同期が有効（または固定）になっている。
- ユーザーが特殊な護石などで個別に調整したい場合に、同期ロジックが強制されることが不具合（あるいは機能欠如による不便）として報告されている。

## 修正内容
1. `ROICalibrator.tsx` で `useROIStore` から `isSyncEnabled`, `setSyncEnabled` を取得する。
2. `items` ステップのサイドバーに、同期の有効/無効を切り替えるトグルスイッチを追加する。
3. トグルの状態に応じて、同期ロジックのオン/オフがユーザーに伝わるような視覚的フィードバック（アイコンやラベルの変化）を提供する。

## 影響範囲
- `src/components/ROICalibrator.tsx`

## 検証
- トグルを OFF にしたとき、あるスロットを移動しても他のスロットが追随しないこと。
- トグルを ON にしたとき、同期ロジックが再び有効になり、移動やギャップ調整が全体に反映されること。
