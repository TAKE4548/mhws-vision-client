# Task: InteractiveCanvas.tsx のレイアウト安定化とフリッカー防止 (TASK-026-02)

## 概要
背景画像更新時のレイアウトシフト（ガタつき）を防止し、スムーズな遷移を実現します。

## ターゲットファイル
- `src/components/roi/InteractiveCanvas.tsx`

## 実装内容
1. `useEffect` (L31付近) で `setActualRatio(null)` を無条件に行うのをやめます。
2. `previewImage` が `null` の際に `actualRatio` をリセットするロジック (L38-41) を、新しい画像がロードされるまで旧比率を維持するように調整します。
3. `imgSrc` が更新される際、新しい画像がロードされるまでコンテナのサイズが変わらないように CSS またはインラインスタイルを固定します。
4. `img` タグの `onLoad` ハンドラで、新しい画像がロードされた後にのみ `actualRatio` を更新するようにします。

## 期待される挙動
- ステップ遷移時、キャンバスの枠が一旦デフォルトサイズに戻るような挙動がなくなること。
- 画像が切り替わる際、旧画像から新画像へスムーズに（または瞬時に）切り替わり、白い空白時間が最小化されること。
