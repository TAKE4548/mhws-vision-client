# TASK-024-02: Vitest の設定統合と TypeScript 型定義の追加

## 概要
Vite の設定ファイルに Vitest 用の設定を追加し、TypeScript がテスト用のグローバル API を認識できるようにします。

## 完了条件
- [ ] `vite.config.ts` に `test` プロパティが追加され、環境として `jsdom`、セットアップファイルとして `./src/test/setup.ts` が指定されている。
- [ ] `tsconfig.json` の `compilerOptions.types` に `vitest/globals` が追加されている。

## 対象ファイル
- `vite.config.ts`
- `tsconfig.json`

## 実装上の注意
- `defineConfig` が Vitest の型を認識できるよう、ファイルの冒頭に `/// <reference types="vitest" />` を追加するか、`vite/config` ではなく `vitest/config` から `defineConfig` をインポートするように変更する。
