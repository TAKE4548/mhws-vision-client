# TASK-020-01: roiStore.ts の同期ロジック実装

## 概要
スロットおよびスキルの全枠を同期させるための基盤ロジックを `roiStore.ts` に実装する。

## 工程
- [ ] `ROIState` インターフェースへの `gaps` 追加
- [ ] `DEFAULT_PROFILE` への `gaps` 初期値設定
- [ ] `updateRelativeRect` の修正
  - カテゴリ（slot_icon, slot_level, skill_name, skill_level）ごとに共通の同期関数 (`syncGroup`) を適用。
  - どのIDが更新されても、グループの基準点を再計算して全3枠を更新するように実装。
- [ ] `setGap` アクションの追加

## 完了条件
- `updateRelativeRect` を呼び出した際、指定したカテゴリの全3枠の座標が、間隔（gap）を維持して一括更新されること。
- Zustand ストアの出力が設計通りの座標値になっていること。
