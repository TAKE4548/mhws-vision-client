### REQ-027: ROI同期グループの細分化（項目別独立設定）
- **Type**: refinement
- **Status**: done (2026-04-23)
- **Current step**: none
- **Priority**: P1
- **Surface**: 「ROIキャリブレーションの3.ITEM ROISで、スロットのレベル枠設定がアイコン設定側と、スキルのレベル枠設定がスキル名側とそれぞれ同期してしまっている。この4種はそれぞれ独立設定で、1種あたり3枠存在するが、その3枠が同期してほしいだけ」
- **Root Cause**: Information Design / Interaction - 同期グループの定義が粗く、異なる種類のUI要素（アイコンとレベル枠、スキル名とレベル枠）が同一の連動ロジックに巻き込まれている。
- **Requirement**: ROI設定の「ITEM ROIS」ステップにおいて、設定項目を4つの独立した同期グループに分離し、各グループ内でのみ連動（等間隔同期）が発生するように改善する。
- **Acceptance criteria**:
  - ROI設定の「ITEM ROIS」ステップに以下の4つの独立した設定カテゴリが存在すること：
    - スロット：アイコン (Slot Icons)
    - スロット：レベル枠 (Slot Level Borders)
    - スキル：名称 (Skill Names)
    - スキルのレベル枠 (Skill Level Borders)
  - 各カテゴリ内の1つ目の要素（例：スロット1のアイコン）を動かした際、同カテゴリ内の他の要素（例：スロット2・3のアイコン）のみが連動して動くこと。
  - カテゴリを跨いだ連動（例：スロットアイコンを動かしたときにスロットレベル枠が動く）が発生しないこと。
