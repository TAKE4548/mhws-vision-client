### REQ-030: 動画解析感度調整（スクロールペース連動）

- **Status**: ready
- **Description**: 動画解析における護石の抽出タイミングを、ユーザーが指定したスクロールペース（秒）に基づいて自動調整し、解析開始UIに設定項目を追加します。

#### Acceptance Criteria (AC)
- [ ] AC-1: [FE] 解析開始UI（AnalysisMonitor）に「スクロールペース（秒）」の入力項目があること
- [ ] AC-2: [FE] 詳細設定（Advanced Setting）から静止しきい値などを直接操作できること
- [ ] AC-3: [FE] 設定値を解析開始API（`startAnalysis`）に正しく渡せること
- [ ] AC-4: [FE] 簡易設定（秒）と詳細設定（しきい値等）が双方向に連動すること
