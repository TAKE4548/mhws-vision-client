### REQ-028: 実機サーバ接続時の動画アップロードおよび解析開始プロセスの修正
- **Type**: defect
- **Status**: done
- **Current step**: none
- **Priority**: P0
- **Surface**: Live Mode（実サーバ接続）において、動画のアップロードが失敗し、解析を開始できない。
- **Root Cause**: 
  - APIクライアントが `multipart/form-data` ヘッダーを境界値（boundary）なしで送信している。
  - OpenAPI 仕様上のタグ不足により、Orval によるクライアントコード生成が一部不完全。
- **Requirement**: 実機サーバ環境において、動画のアップロード、SSE接続、解析開始の一連のフローがスタブなしで完遂できること。
- **Acceptance criteria**:
  - ユーザーが選択した動画ファイルが、実サーバのエンドポイントに対して正常にアップロード（202 Accepted）されること。
  - アップロード後、実サーバから発行されたジョブIDを用いて SSE 接続が確立されること。
  - 「INITIATE ANALYSIS」ボタン押下時、実サーバに対して解析開始リクエストが正常に送信されること。
  - Orval による API クライアント生成がエラーなく実行され、すべての必要なメソッドが利用可能であること。
- **Design doc**: none
