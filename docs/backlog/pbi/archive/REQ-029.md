### REQ-029: 解析済み護石のキャプチャ画像表示機能の復旧

- **Status**: ready
- **Surface**: "解析後のカードに本来表示されるはずのクロップ画像が表示されない。詳細画面に入っても表示されない。コンソールにもGETでエラーが出てる。"
- **Root Cause**: **Interaction** - システムが解析結果の視覚的エビデンス（クロップ画像）を提供できていないため、ユーザーによる判定結果の目視確認が不可能になっている。
- **Requirement**: 護石の解析精度をユーザーが目視で検証できるように、解析に使用された各護石のキャプチャ画像をUI上で閲覧可能にする。

#### Acceptance Criteria (AC)
- [x] AC-1: 解析結果一覧のカードに、該当する護石のクロップ画像が表示されていること。
- [x] AC-2: 解析結果詳細画面に、該当する護石のクロップ画像が表示されていること。
- [x] AC-3: 画像取得時のネットワークエラー（GETエラー）が解消されていること。
- [x] AC-4: 動画解析時、全ての解析済み護石に対して画像が正しく紐付けられていること。
